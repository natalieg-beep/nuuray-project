# Nuuray Glow

Kosmische Unterhaltung — Horoskope, Tagesenergien, Mondphasen, Partner-Check.

## Setup

```bash
# Im apps/glow Verzeichnis:
flutter pub get
flutter run
```

## Dependencies (Shared Packages)

```yaml
dependencies:
  nuuray_core:
    path: ../../packages/nuuray_core
  nuuray_api:
    path: ../../packages/nuuray_api
  nuuray_ui:
    path: ../../packages/nuuray_ui
```

## MVP Features

1. Auth (Email + Apple + Google Sign-In)
2. Onboarding: Geburtsdaten eingeben
3. Tageshoroskop-Ansicht
4. Mondphasen-Kalender
5. Wochen/Monatsüberblick
6. Partner-Check
7. Premium-Gating + In-App Purchase
8. Push-Notifications
